# E2E联调测试目录

本目录用于存放前端 E2E 联调测试执行脚本、环境启动脚本与说明材料。

当前 E2E 场景覆盖：

- 未登录访问受保护页面时跳转登录，并在登录后回跳到目标页面
- 管理员登录、访问系统管理、退出登录完整链路
- 档案员、借阅人、审计员等内置角色的菜单范围与受限页面拦截
- 工作台待办明细直达审批、出库、归还操作
- 文件访问权限正负向场景，包括预览、下载、密级脱敏与审计留痕
- 借阅闭环与销毁闭环的跨角色联调场景
- 通知中心 `notificationId` 聚焦、销毁审批通知回跳、报表导出通知回跳
- 系统管理 `tab/userId` 深链、档案 `archiveId/fileId` 深链、扫描任务 `itemId` 深链

执行方式：

```bash
./tests/e2e_tests/run_e2e_tests.sh
```

或：

```bash
./scripts/run_tests.sh e2e
```
