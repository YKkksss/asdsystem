# 单元测试目录

本目录用于存放单元测试执行清单、执行脚本及相关说明材料。

当前单元测试重点覆盖以下内容：

- 认证登录失败锁定、刷新令牌校验等核心认证逻辑
- 档案状态流转、密级访问判断、修订字段变更计算
- 借阅超期计算、提醒类型判定、超期状态同步、非法申请校验
- 扫描任务状态汇总、扫描任务合法性校验、上传文件格式与大小校验
- 系统初始化命令的幂等性与默认账号生成逻辑

执行方式：

```bash
./tests/unit_tests/run_unit_tests.sh
```

或在 `fullstack/` 目录统一执行：

```bash
./scripts/run_tests.sh unit
```
